题目名称：有效的山脉数组

难度：⭐

题目描述：

给定一个数组A，如果它是有效的山脉数组就返回true，否则返回false。

让我们回顾一下，如果A满足下述条件，那么它是一个山脉数组：

  • A.length >= 3

  • 在 0 < i < A.length - 1 条件下，存在 i 使得：

    • A[0] < A[1] < ... A[i-1] < A[i]
  • A[i] > A[i+1] > ... > A[A.length - 1]
