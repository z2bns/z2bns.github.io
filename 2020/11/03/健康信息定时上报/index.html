前言

思路

然后再服务器上使用crontab设置定时执行python程序

crontab常用命令：

  1. 查询当前系统用户设置了哪些执行任务：crontab -l
  2. 清空当前系统用户设置的所有任务：crontab -r
  3. 编辑和设置当前系统用户要自动执行的任务：crontab -e
  4. 检查crontab任务执行日志：cat /var/log/cron
