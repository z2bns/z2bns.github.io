总结：

没有官方题解，不过题解里有一些好的思想，从位运算进行求解。

与运算逐位判断：

  • 判断 n 最右一位是否为 1 ，根据结果计数。
  • 将 n 右移一位，循环。

巧用n&(n-1)：

  • (n - 1) 解析： 二进制数字 n 最右边的 1 变成 0 ，此 1 右边的 0 都变成 1 。
  • n \& (n - 1)解析： 二进制数字 n 最右边的 1 变成 0 ，其余不变。

多种解法解题效率差不多。
