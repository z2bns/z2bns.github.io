红警快捷键

发表于 2021-08-13 分类于 娱乐
| 阅读次数： |
Valine：
本文字数： 837 阅读时长 ≈ 1 分钟

红色警戒2战网对战平台，可以在线对战，😙

红色警戒常用快捷键如下：

1、部署对象 / 部队 D

2、警戒目前区域 G

3、移动攻击 点选部队 按下 Ctrl + Shift + 点选部队或区域

4、散开 X

5、强制开火 按住 Ctrl 键，用光标指示目标，再左击

6、强制移动 按住 Alt 键，用光标指示目标，再左击

7、选项选单 ESC

8、编队 Ctrl + 1 ~ 9

9、与选定阵营同盟 A

10、选择同类型部队 T

11、与所有观战者交谈 （多人模式） 按 Enter 开启交谈功能，输入讯息后，再按 Enter 传送讯息，按鼠标右键取消讯息。

12、与所有盟友交谈（多人模式） 按 Home 键开启交谈功能，输入讯息后，再按 Enter 传送讯息，按鼠标右键取消讯息。

13、与所有玩家交谈（多人模式） 按 End 键开启交谈功能，输入讯息后，再按 Enter 传送讯息，按鼠标右键取消讯息。

14、布置信标 按下 B 键，按下 Enter 输入讯息后，再按 Enter 传讯，按 Del 取消讯息。

15、进入路径点模式 点选部队 按住 Z 键，设定路径点，放开 Z 键开始让部队照路径前进。

16、设定集合点 点选兵营、战车工厂、造船厂或复制中心，再于战场上选择集合点。

17、前往雷达事件地点 空格键

18、所有部队喝采 C

19、开启外交选单 Tab

20、跟随镜头 F

21、守卫指定点 Ctrl + Alt + 点选区域

22、护卫部队 Ctrl + Alt + 点选部队

23、防守建筑 Ctrl + Alt + 点选建筑

24、建筑页面 Q

25、武器页面 W

26、步兵页面 E

27、车辆页面 R

28、下一个部队 M

29、上一个部队 N

30、全选 P

31、以精锐为准选择部队 Y

32、以生命值为准选择部队 U

33、调整已编队部队 按住 Shift 键，点选已选定部队取消选取或点选未选取的部队，将之加到编队中。

34、将战略地图以基地为中心置中 H

35、维修模式 鼠标左点建筑，K

36、变卖模式 鼠标左点建筑，L

37、设立书签 Ctrl + F1 ~ F4

38、前往书签所在位置 F1 ~ F4

39、多人联机叫嚣语言 F5 ~ F12

40、游戏中截图 Ctrl + C （所截图片在红警程序文件夹里可以找到）
